# 作者使用指南

本指南帮助 `author` 角色完成账号注册、作品上传与后续维护。

## 1. 注册与登录
1. 在 `/author/register` 填写：
   - 姓名、邮箱（作为登录账号）、密码。
   - 学校、学院、界别、学号（用于参会统计）。
2. 注册成功后会自动跳转登录页，可用刚设置的邮箱/密码登录。
3. 登录后导航栏会显示“作者中心”“上传作品”等入口。

## 2. 作者中心（`/author/profile`）
进入后可看到：
- 个人信息卡片：展示注册时填写的基本信息。
- “我的投稿”表格：列出所有作品的标题、方向、Track、审核状态（待审核/通过/未通过）、投稿状态（中稿/已发表）。
- 常用操作：
  - “上传作品”按钮直达投稿页。
  - “编辑”跳到 `/author/submissions/:id/edit`，可修改信息、重新上传海报。
  - “撤稿”会删除该投稿（需确认），适合撤销错误提交。
  - “查看详情”链接可跳到公开作品页，确认展示内容。

## 3. 投稿/编辑表单
路径：`/author/submissions/new` 或 `/author/submissions/:id/edit`。

必填字段：
- **论文标题 / 摘要 / 接受会议或期刊 / 联系方式**。
- **Track**：Poster 或 Demo，可随时切换。
- **投稿状态**：`中稿` or `已发表`（映射 `accepted/published`）。
- **投稿方向**：从管理员维护的 `Direction` 列表中选择；若后续想调整，可在编辑页重选。
- **海报文件（可选）**：上传 PDF/图片以供展板使用；编辑时如未重新上传，会沿用旧海报。
- **档案授权**：`archive_consent` 默认为 `true`。
- **Paper URL**：可填写对外链接，便于浏览者下载。

提交成功后页面会提示“提交成功”，并在 0.8s 后导航回作者中心。

## 4. 审核与展示
- 审核状态在作者中心实时更新：管理员通过 `/admin/exhibits` 审核。
- 作品通过后会获得序号，可在 `/papers/:id` 页面查看投票与奖项标签。
- 如需修改信息，请在评审前完成；审核后仍可编辑，但建议先与管理员同步。

## 5. 常见问题
- **忘记密码**：联系管理员重置（当前无邮箱找回流程）。
- **方向缺失或错误**：通知管理员在“后台管理 → 审稿方向”新增或更正，再回到投稿页编辑。
- **附件过大**：建议压缩后再上传；文件实际存储于服务端 `data/uploads/`，可通过后台下载接口验证。

若需了解评审与奖项流程，可阅读 `docs/ROLE_REVIEWER.md` 与 `docs/ROLE_ADMIN.md`。
