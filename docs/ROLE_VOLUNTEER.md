# 志愿者使用指南

面向注册为 `volunteer` 角色的用户，涵盖报名、信息查询、报销与账号维护。

## 1. 报名与登录
1. 打开 `/volunteer/register`，填写以下信息：
   - 基本信息：姓名、邮箱、学校、学院、界别、学号（可选但建议填写）。
   - 志愿偏好：志愿方向（可多选）、可服务时段（可多选）、是否愿意参与计票。
   - 学院字段会作为后续联系人信息，确保准确。
2. 提交后系统会立即创建账号，可前往 `/login` 使用邮箱 + 密码登录。

## 2. 个人信息页（`/volunteer/profile`）
- 展示管理员配置的组织职责、志愿方向、可服务时段、计票报名状态等。
- 若管理员在后台为你分配了多个组织，会逐条列出 `组织名：职责描述`。
- 页面底部提供“修改密码”表单：输入当前密码、新密码及确认后即可生效。

## 3. 报销管理（`/reimbursements`）
### 3.1 创建/编辑
- 表单字段：项目名称、组织（默认选中所属组织，可选择管理员分配的任一工作组）、报销内容、数量、金额、发票抬头公司、附件（可选，支持 PDF/图片）。
- 提交后记录初始状态为“待审核”，你可以继续编辑或删除，直到管理员审批。
- 附件会展示为链接，可直接下载确认。

### 3.2 列表与状态
- 列表按状态分类：待审核、通过、拒绝、等待补充。
- 点击记录可展开详情、查看管理员备注。
- 若管理员要求补充材料，可重新编辑并重新上传附件。

### 3.3 常用操作
- “导出报销”按钮仅管理员可见；普通志愿者仅能管理自己的申请。
- 表单支持草稿式操作：切换状态不会丢失已输入数据，除非点击“重置”。

## 4. 计票志愿者
- 报名时勾选“愿意参与计票”后，管理员会在后台审批。
- 一旦获批（分配了带有 `can_edit_vote_data` 的角色模板），在公开成果页可看到投票编辑入口；否则仍以普通志愿者权限展示。

## 5. 账号与安全
- 如果忘记密码需联系管理员重置（当前无邮箱找回逻辑）。
- 登录信息保存在浏览器 `localStorage` 的 `token` 字段，更换设备需重新登录。
- 退出登录请使用导航栏“退出”按钮，避免他人使用你的会话。

如需了解后台如何使用你的报名信息，可参考 `docs/ROLE_ADMIN.md` 或直接与组织者联系。
