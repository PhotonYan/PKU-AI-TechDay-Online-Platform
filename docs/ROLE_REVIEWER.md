# 审阅者使用指南

本指南适用于拥有 `reviewer` 角色的老师或评审，覆盖邀请码注册、登录及奖项推荐流程。

## 1. 获取邀请码
- 由管理员在“后台管理 → 审阅者邀请码”生成 8 位邀请码，可选预设审阅方向。
- 邀码仅可使用一次，注册成功后自动标记为已使用。

## 2. 注册步骤（`/reviewer/register`）
1. 输入姓名、邮箱、邀请码、密码。邀请码可点击“验证”按钮提前校验：若有效且绑定方向，会自动锁定该方向。
2. 选择审阅方向（若邀请码已预设则不可更改）。邮箱将作为登录账号，请填写真实可用地址。
3. 提交后提示“注册成功”，系统自动跳转登录页。

## 3. 登录与导航
- 登录入口与普通用户一致：在 `/login` 输入注册邮箱和密码即可。若忘记邮箱，请联系管理员在后台查看“审阅者邀请码管理”，那里会显示已注册的邮箱。
- 登录后导航栏会显示“奖项管理”入口，同时保留成果展示、退出等菜单。
- 若管理员还分配了其他角色（例如志愿者），相应页面也会出现。

## 4. 奖项管理页（`/awards`）
### 4.1 列表与筛选
- 默认显示 Poster Track，可切换至 Demo。
- 审阅者会自动限定在注册时的方向，无法查看其他方向作品。
- 右上角可按序号/投稿编号排序；筛选条能根据状态（无标签、已推荐、具体奖项）筛查。

### 4.2 推荐操作
- 点击某作品行的“推荐”按钮打开对话框，填写：
  - 推荐理由（必填）。
  - 信心度（0~1，可选，支持小数）。
- 保存后列表中的“推荐”列会显示“1 条推荐”链接，点击可查看自己与其他审阅者的建议。
- 可随时重新编辑，后端只保留最新记录。

### 4.3 查看详情
- 标题列提供“查看详情”链接（新窗口），可审阅公开展示页的海报和投票情况。
- 奖项标签列会显示管理员已授予的奖项彩色标签，便于对照官方结果。

## 5. 注意事项
- 审阅者无权直接授奖，只能提交推荐供管理员参考。
- 若需要修改方向或重置密码，请联系管理员在后台“用户列表”中调整。
- 推荐记录通过 `/api/awards/recommendations` 读写，修改会被立即应用，无需额外保存。

更多关于管理员评奖与后台配置的说明请阅读 `docs/ROLE_ADMIN.md`。
